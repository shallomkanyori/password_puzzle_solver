About

This project is my portfolio project for Holberton School, specifically the ALX SE program. It was inspired by my love for games especially puzzles. In this innovative take on the classic game Hangman, the premise is to figure out the password in 7 tries or less while challenging yourself to beat your best score. It is a single-player free game with no login required. This game is intended for entertainment purposes only, with no ties to hacking or cybersecurity.
